# MealSnap Design Reference

## 🎨 Color Palette

- **Primary Green:** #43A047  
  _A fresh, vibrant green that evokes health, growth, and natural foods._
- **Accent Orange:** #FFB300  
  _A warm, energetic orange for highlights and call-to-action buttons._
- **Background Light:** #F9FBE7  
  _A soft, light background for a clean, inviting look._
- **Neutral Gray:** #757575  
  _For text and subtle UI elements._
- **Error Red:** #E53935  
  _For error states and warnings._

## 🖌️ Usage Guidelines

- Use **Primary Green** for main branding, headers, and key actions.
- Use **Accent Orange** for buttons, highlights, and important UI elements.
- Use **Background Light** for page backgrounds.
- Use **Neutral Gray** for secondary text and icons.
- Use **Error Red** for error messages and destructive actions.
